### Example: Game Character Creation System

### Scenario

Imagine a simple game where characters have:

- name

- health

- speed

- weapon

- skinColor

You have predefined character presets:

- Warrior

- Archer

- Mage

When creating a new character:

- The name changes

- Maybe health or weapon changes

- Most properties stay the same
